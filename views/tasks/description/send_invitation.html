Einladungen versenden

Vor der Anmeldung zu einem Marktereignis wird an jede bekannte Person eine persönliche Einladung versandt. Dabei ist unerheblich, ob oder wann die Person tatsächlic an einem Markt teilgenommen hat. Diejenigen, die ihren E-Mail-Kontakt deaktiviert haben, werden dabei natürlich ausgelassen.
